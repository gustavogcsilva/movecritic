# MoveCritic 🎬📚

O **MoveCritic** é uma plataforma interativa desenvolvida em **Python** utilizando **Flask** para a criação de uma API robusta, além de uma interface web dinâmica. O projeto foi projetado para amantes de cinema e literatura que desejam pesquisar informações detalhadas sobre seus filmes e livros favoritos, registrar opiniões pessoais e catalogar suas resenhas em um só lugar.

---

## 🚀 Funcionalidades

* **Busca Integrada:** Pesquise por títulos de filmes e livros e obtenha dados detalhados sobre as obras.
* **Sistema de Avaliações (Reviews):** Escreva, edite e organize seus comentários e notas de forma personalizada.
* **Persistência de Dados:** Integração com banco de dados para salvar o histórico de críticas e perfis de usuários de maneira segura.
* **Interface Responsiva:** Frontend limpo e fluido estruturado para uma navegação agradável tanto no desktop quanto no celular.
* **API RESTful:** Arquitetura de backend modular usando Flask, permitindo fácil integração com outros serviços.

---

## 🛠️ Tecnologias Utilizadas

* **Backend / API:** [Python](https://www.python.org/) & [Flask](https://flask.palletsprojects.com/)
* **Frontend:** HTML5, CSS3 & JavaScript (Design responsivo e interações limpas)
* **Banco de Dados:** MySQL (para persistência de usuários, críticas e mídias catalogadas)
* **Controle de Versão:** Git & GitHub
